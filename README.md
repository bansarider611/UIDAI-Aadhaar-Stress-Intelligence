# UIDAI Aadhaar Stress Intelligence

This repository contains the analysis developed for the UIDAI Hackathon.

## Objective
To analyse Aadhaar demographic and biometric update patterns and identify
regions experiencing life-event driven and system-driven stress.

## Structure
- notebook/: Jupyter notebook used for analysis
- outputs/: Generated charts and visualisations
- data/: Aggregated datasets (raw data excluded)
- reports/: Final PDF report

## Tools
Python, Pandas, Matplotlib, Jupyter Notebook
